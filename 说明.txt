1.创建容器
docker run -d -ti \
  -v /tmp/.X11-unix:/tmp/.X11-unix \
  -e DISPLAY=unix$DISPLAY \
  -e GDK_SCALE \
  -e GDK_DPI_SCALE \
  --name wine \
  rdvde/wine

2.运行容器
docker exec -ti wine /bin/bash
或者
docker restart wine && docker exec -ti wine /bin/bash
3.执行需要运行的应用
QQ
TIM
WeChat
BaiduNetDisk
ThunderSpeed
Foxmail

提示No protocol specified错误，请在主机运行：
xhost +
